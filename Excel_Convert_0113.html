📌 v2 개선 사항

🔧 이번 버전의 수정 사항:

  • ✅ 파일 첨부 기능 완전 수정 - 클릭 및 드래그 앤 드롭 정상 작동
  • ✅ 데이터 행 인식 문제 해결 - skipEmptyLines: true로 설정하여 모든 행 정확히 인식
  • ✅ 탭(Tab) 구분자 우선 인식
  • ✅ 실시간 디버깅 정보 표시 (파일별 행 수, 컬럼 수, 구분자 종류)
  • ✅ 중복 헤더 자동 처리 (예: Serial Number → Serial Number (2))
  • ✅ 다중 검색 필터 - AND 조건으로 순차 필터링

📝 사용 방법:

  • 업로드 영역을 클릭하거나 파일을 드래그하여 CSV/TXT 파일을 업로드합니다
  • 디버깅 정보를 확인하여 각 파일이 정확히 파싱되었는지 확인하세요
  • 여러 검색 필터를 추가하여 데이터를 정밀하게 필터링할 수 있습니다
  • 필터링된 결과를 Excel, CSV로 다운로드하거나 클립보드에 복사하세요
